Called Click to call plugin allows you to easily issue a clickToCall service between two phone numbers.

== Description ==

This service allows you to easily issue a clickToCall service between two phone numbers. Both calls are dialed, and upon answer, they are bridged together.

You will have to get registered with called.in website to get an API to use this plugin. You will get 10 credits free initially and after that you can buy credits from the website to continue the service.  

**Why use click-to-call?**

Need to connect your potential-customers with sales representatives through the web? Or connect your members to each other over the phone without revealing their phone numbers? Our Click-To-Call API can easily handle your needs!

Supported Countries

The countries currently supported are following : United States, Canada, United Kingdom, Australia, New Zealand, India

**how to you use this plugin?**

    [called] use this shortcode anywhere on your post or pages.

There is a called widget also created which can be used in any widget area.



Thanks :) 


== Installation ==

1. Upload `called` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[called]` shortcode inside your post or pages or you can use the called widget at any place in your theme.
